## ✅ 프로젝트 회고
짧은 기간이었지만 ResNet50, DINOv2를 다뤄 볼 수 있어서 좋았다.
처음 선정한 모델에서 원하는 결과를 얻지 못해서 아쉬움이 남지만
문제를 해결하기 위해 데이터를 분석해본 과정도 기억에 남고
짧은 기간동안 목표를 바꿔서 이마 이미지에 집중하여 결과는 낸 부분에 의미를 두며 다음에는 LLM Fine Tuning에 집중해보고 싶다.
